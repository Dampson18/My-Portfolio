# My-Portfolio - Kelvin Dampson

## Project Overview
This is the personal portfolio of Kelvin Dampson, a dedicated Frontend Developer based in Ghana. The portfolio showcases various web development projects, skills, services, and certifications, providing an intuitive and engaging experience for visitors.

## Features
- **Responsive Design:** Optimized for various devices and screen sizes.
- **Interactive Project Showcase:** Highlights a variety of web applications and designs.
- **Smooth Navigation:** Easy access to different sections with smooth scrolling.
- **Contact Form:** Integrated with EmailJS for direct communication.
- **Testimonial Carousel:** Displays feedback from clients.
- **Resume Download:** Easy access to professional credentials.

## Directory Structure
The project assets have been organized into a dedicated `assets/` directory:
- `assets/css/`: Contains the project's stylesheets.
- `assets/js/`: Contains the JavaScript files for interactivity.
- `assets/images/`: Stores all images, icons, and backgrounds used in the portfolio.
- `assets/docs/`: Holds documents like the resume and other PDF files.

## Technologies Used
- **HTML5:** For semantic and structured content.
- **CSS3:** For styling, including Flexbox and CSS Grid for layouts.
- **JavaScript (ES6+):** For dynamic behavior and interactivity.
- **EmailJS:** For handling contact form submissions.
- **Font Awesome:** For scalable vector icons.
- **Google Fonts:** For modern and readable typography.

## How to Run
Simply open `index.html` in any modern web browser to view the portfolio.
